# 🧮 Calculadora de IMC

Uma calculadora simples de IMC (Índice de Massa Corporal).

## 🚀 Funcionalidades

- Entrada de peso e altura
- Cálculo automático do IMC
- Classificação de acordo com a tabela da OMS
- Layout responsivo e estilizado
